# 客户端

## 用户登录指令

```
login [username] [password]
register [username] [password]

如：register dph dphpassword
```

## 聊天室相关指令

```
列出已加入的聊天室 lstroom
选择已加入的聊天室 chroom [roomid]
申请加入聊天室    joinroom [roomid]
创建聊天室        createroom [roomname]
锁定聊天室        lockroom [roomid]
解锁聊天室        unlockroom [roomid]
```

## 聊天相关命令

在已选择聊天室的继承上执行

```
发送消息    send [msg...]
```

## 其它命令

```
退出       quit
```
